# Installation

## Requirements

Needs fedora 6.1 One-Click-Run running on port 8082 
  from here: https://wiki.lyrasis.org/display/FEDORAM6M1P0/Quick+Start


## Startup

1. Launch fedora 6.1 One-Click-Run (on port 8082)!
2. Launch spring boot app afterwards
